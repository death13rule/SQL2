# Домашнее задание к занятию "Репликация и масштабирование. Часть 2" - `Лютиков А.В.`

### Задание 1
Опишите основные преимущества использования масштабирования методами:

активный master-сервер и пассивный репликационный slave-сервер;
master-сервер и несколько slave-серверов;
Дайте ответ в свободной форме.

### Решение 1
Активный master-сервер и пассивный репликационный slave-сервер:
Преимущества:
1. Отказоустойчивость
2. Если master выходит из строя, slave можно быстро переключить в master
3. Резервирование данных
4. Все данные копируются на slave практически в реальном времени.
5. Минимизация downtime
6. При аварии время простоя сокращается.
7. Бэкапы без нагрузки на master
8. Резервные копии можно делать со slave.
9. Простота реализации
10. Самая простая схема репликации.
Недостаток:
1. Slave обычно простаивает или используется мало.

Master-сервер и несколько slave-серверов:
Преимущества:
1. Масштабирование чтения
2. SELECT-запросы распределяются между slave.
3. Разгрузка master
4. Master занимается только INSERT/UPDATE/DELETE.
5. Повышение производительности
6. Гибкость
7. Один slave можно использовать для аналитики, другой для резервного копирования.
8. Отказоустойчивость
9. При падении одного slave система продолжает работать.
Недостаток:
1. Запись ограничена одним master.
